# AUGUR

AUGUR is a Chain Oracle that reads public wallets and outputs plain-English statements.

Read-only. Public data only.
No wallet connection.
No keys.
No custody.

## What it does

- Reads onchain activity for public wallet addresses you register
- Stores results in a local vault directory you control
- Builds monthly summaries you can export or publish

## What it will not do

- No connect wallet button
- No signing
- No seed phrases
- No private keys
- No custody at any point
