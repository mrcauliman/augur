Digital Ledger — Runbook

DL v1 is a local-first vault that ingests public on-chain account history and generates snapshots, events, and monthly summaries. The vault is the product. The API is the engine.

repo/var/www/digital-ledger
apiapps/api
vaultvault/sample_vault
port8787

Core concepts

  • Accounts are wallets or identifiers that we track
  • Snapshots are point-in-time balances per account
  • Events are transaction-derived ledger movements used for reporting
  • Indexes track last_seen and tx dedupe to avoid re-ingesting the same history

Install and dev start

cd /var/www/digital-ledger
corepack enable
corepack prepare pnpm@9.0.0 --activate

Accounts

List accounts
curl -sS http://127.0.0.1:8787/api/accounts
Add XRPL account
curl -sS -X POST http://127.0.0.1:8787/api/accounts \
  -H 'content-type: application/json' \
  -d '{"type":"onchain_wallet","chain":"xrpl","label":"public-hot:ripple-11","address_or_identifier":"rB3WNZc45gxzW31zxfXdkx8HusAhoqscPn"}'
Pause account
curl -sS -X POST http://127.0.0.1:8787/api/accounts/ACCOUNT_ID/pause
Resume account
curl -sS -X POST http://127.0.0.1:8787/api/accounts/ACCOUNT_ID/resume

Expected behavior

  • First run may ingest a chunk of history and write event files
  • Subsequent runs should often fetch 0 because nothing new happened

Reset controls

If you need to reingest history for an account, wipe its dedupe keys and last_seen entry.

cd /var/www/digital-ledger

cp -a vault/sample_vault/data/indexes/tx_dedupe.json vault/sample_vault/data/indexes/tx_dedupe.json.bak.$(date +%s)
cp -a vault/sample_vault/data/indexes/last_seen.json vault/sample_vault/data/indexes/last_seen.json.bak.$(date +%s)

python3 - <<'PY'
import json, pathlib
account_id = "acct_xrpl_2df93d1e"
prefix = f"xrpl:{account_id}:"
k_last = f"xrpl:last_ts:{account_id}"

p = pathlib.Path("vault/sample_vault/data/indexes/tx_dedupe.json")
d = json.loads(p.read_text())
d2 = {k:v for k,v in d.items() if not k.startswith(prefix)}
p.write_text(json.dumps(d2, indent=2, sort_keys=True))
print("dedupe_removed", len(d)-len(d2))

p2 = pathlib.Path("vault/sample_vault/data/indexes/last_seen.json")
ls = json.loads(p2.read_text())
removed = 1 if k_last in ls else 0
ls.pop(k_last, None)
p2.write_text(json.dumps(ls, indent=2, sort_keys=True))
print("last_seen_removed", removed)
PY

pnpm snapshot
