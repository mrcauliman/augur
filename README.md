# AUGUR

AUGUR is a Chain Oracle that reads public wallet history and produces plain-English statements.

Read-only. Public data only.
No wallet connection. No keys. No custody.

## What it does

- Tracks public addresses you provide
- Pulls explainable history
- Produces clean snapshots and monthly summaries
- Outputs machine-readable data you can archive and audit

## What it does NOT do

- No connect wallet button
- No signing
- No seed phrases
- No private keys
- No custody at any point
