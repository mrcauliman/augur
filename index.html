what it is
DL is not an exchange, not a wallet, not a tracker site. It is a private ledger you control. It stores normalized data in a vault folder and produces reports you can read, export, and automate.

v1 capabilities
  • Account registry in vault/sample_vault/data/accounts/accounts.jsonl
  • Snapshots per chain written to data/snapshots
  • Event stream for XRPL written to data/events
  • Dedupe and last seen indexes in data/indexes
  • Monthly report generator to reports/monthly
why it wins
  • One vault format across chains
  • Local first and portable
  • Report engine is deterministic, easy to audit
  • No third party dashboard dependency
